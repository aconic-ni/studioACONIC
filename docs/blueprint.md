# **App Name**: Page Canvas

## Core Features:

- Blank Page: A blank canvas page to which components can be added
- Add Component: Add components to the blank canvas page.
- Drag and Drop Components: Components should be movable via drag and drop on the canvas.
- Robust component framework: Use React components for building robust UI

## Style Guidelines:

- Primary color: Soft blue (#A0CFEC), evoking calmness and space for creation.
- Background color: Very light gray (#F0F0F0), providing a neutral backdrop that doesn't distract from the content.
- Accent color: Muted teal (#62B1A7), to highlight interactive elements and call to action buttons.
- Font: 'Inter', sans-serif, for a modern and neutral feel, suitable for both headlines and body text.
- Simple, clean icons for actions like adding or moving components.
- Clean and open layout, providing plenty of space for arranging components.
- Subtle transitions and animations for dragging and dropping components.
